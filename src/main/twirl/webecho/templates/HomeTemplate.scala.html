@context.page.appcode service allows you to easily define on the fly JSON recorders which can be fed by remote services using either the dynamically generated webhook or the websocket you've provided. Then, at any time you can check what have been sent by the configured remote services thanks to the provided API.

This service currently manages @context.stats.count json recorders.

Take a look to the Swagger API specification to use this service or go to the @context.page.appcode github repository if you want to contribute.
